# RR_ROBOT
### software requirement:

1. Ubuntu 20.04
2. ROS (Robot Operating System) :  version 1.16.0 
3. Arduino IDE : version 1.8.15 
4. Rosserial Arduino Library 0.7.9
5. Visual Studio Code :  version 1.86.2 
6. Tkinter GUI python library
